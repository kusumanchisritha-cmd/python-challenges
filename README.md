# Energy Efficiency Program

## Description

This program analyzes energy readings and classifies them into efficient, moderate, high, and invalid categories.

## Objective

To check energy usage and generate a simple efficiency report.

## Features

* Classifies readings
* Calculates total energy
* Counts buildings
* Shows final result

## Concepts Used

Lists, loop, conditions, dictionary, list comprehension, tuple

## Sample Input

[45, 120, 200, 30, 75, 180, -5, 60, 15, 250]

## Sample Output

Efficient: [45, 30, 15]
Moderate: [120, 75, 60]
High: [200, 180, 250]
Invalid: [-5]

Total: 975
Buildings: 9

Result: Balanced Usage, Energy Waste

## Conclusion

The program helps in understanding energy usage using Python.
